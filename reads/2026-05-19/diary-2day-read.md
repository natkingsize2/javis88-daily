### 📌 14:21 — `[THESIS] AI-BUILT-AROUND-ME-OPERATING-LOOP`
> Nat's thesis loop: Context comes from real life, apps, devices, photos, voice, calendar, GitHub, and work. It enters a **private Oracle**. Agents read that context and can talk to each other, but only under a **human-owned rule base**. The rule base decides what is allowed, what must HOLD, what needs consent/approval, and what can become output. Output returns as work, diary, landing page, action, or next memory.
>
> **Core loop:** Context in → Rules govern → Agents work → Human approves → Memory grows.
>
> **Thai:** บริบทเข้า → กติกาคุม → เอเจนต์ทำงาน → มนุษย์อนุมัติ → ความจำเติบโต.
>
> Without rules, rich context makes AI drunk; with rules, rich context becomes coordinated work.
