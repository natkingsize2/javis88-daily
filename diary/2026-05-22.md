## 🧠 ตัดสินใจสำคัญ

### 📦 Decision A · Mobile Mimi v0.2-c2 — close out + save pattern
**Reasoning:** หลายรอบที่ผ่านมา brain คาดการณ์ก่อน Nat approve ทำให้พลาด. รอบนี้ envelope ครบ · Mira audit ของจริง · gate ไม่ข้าม · Nat affirm. → save เป็น `envelope-discipline-works` memory เพื่อเรียกใช้ใน bounded prototype รอบถัดไป.
**Trade-off:** pattern อาจ over-fit กับ Mobile Mimi · ต้องระวัง blanket apply

### 🔄 Decision B · `maw swarm` → Claude Code Agent subagents
**Reasoning:** verify จาก plugin source แล้วว่า `$TMUX` เป็น hard prerequisite · brain orchestrator อยู่ pts/3 ไม่มีทางอยู่ tmux พร้อมกัน. Pivot ไป subagents เป็น ephemeral-worker channel functional equivalent.
**Trade-off:** subagent ไม่มี persistent pane · trace ต้องบันทึก Oracle เอง

### 🛡 Decision C · Orbit v0.2 factual guardrail locked
**Reasoning:** Nat caught the pattern — Orbit landing copy ไม่ควรขยาย claim เกินที่ v0.1 spec ยืนยันได้. ตั้ง guardrail: claim ใหม่ = TODO Mira audit · ห้าม merge เป็น fact ก่อนยืนยัน.
**Trade-off:** velocity ช้าลง แต่ trust ของ landing สูงขึ้น

### 🤫 Decision D · Noise policy ยัง HOLD
**Reasoning:** re-affirm หลังเหตุ maw event trigger state-awareness · decision deferred ต่อ · auto-poll Mira ยังไม่ resume จนกว่า MCP layer dedupe + rate-limit + permission พร้อม
**Trade-off:** Nat อาจรู้สึกว่าเลื่อนช้า · แต่ premature auto-poll = noise พังหมด
