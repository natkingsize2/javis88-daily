**เหตุผล:** 2 อาทิตย์ลังเลก็ได้แค่ความลังเล · lock = ตัด option · ตัด option = ทำงานเร็ว · Anthropic for Startups ต้องการ mission ชัด ไม่ใช่ pitch ลอย

### 🎯 Decision 2 · Lock goal · 1000 subs by Dec 31, 2026
**ก่อน:** "ขอแค่ venture ไปได้ก็พอ"  
**หลัง:** "1000 active subscribers · Dec 31, 2026 · path to Anthropic for Startups"  
**เหตุผล:** Goal คือ filter · ทุกงานวันนี้ตอบได้ว่า "ทำเพื่อ 1000 subs?" — ถ้าตอบไม่ได้ · don't do it · 7 เดือนครึ่งจาก now

### 🚀 Decision 3 · Ship plugin docs ก่อน PR #86
**Context:** 4 plugins (anchor-check · fleet-reap · spec-lint · session-summary) ต้องมี USE-CASES.md + README Why ก่อน registry submit  
**เหตุผล:** Registry คือ shopfront · คนเข้ามาเจอ plugin ที่ไม่มี doc = ผ่านไป · 131-138 lines/file + real transcript = พร้อมขาย

### 💻 Decision 4 · Provision mini-muscle เป็น worker node คืนนี้
**ก่อน:** mini-muscle = old Windows machine นั่งเฉยๆ  
**หลัง:** Ubuntu 24.04.4 LTS · Tailscale · Bun · Node · Claude · brain ssh ได้  
**เหตุผล:** sgp1 1 machine ไม่พอ scale agent team · บ้านมี mini-muscle อยู่แล้ว · เลิก Windows · gain 1 tier worker · cost = 1 คืนกับ Ventoy
