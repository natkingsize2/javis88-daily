### Orbit ขึ้นบินจริง — เลขาเด็กพูดไทยได้ในคืนเดียว
**7 PR รวมร่าง · เสียงสังเคราะห์ edge-tts · แต่บรรณาธิการสารภาพ: "ผมแก้เองทั้งที่ควรสั่งทีม"**

กรุงเทพฯ — ในมาราธอนยาวกว่า 36 ชั่วโมงที่ปิดท้ายเมื่อคืนวาน กองเรือของ Nat ทำสิ่งที่ค้างมานานให้จบในรอบเดียว: โปรเจกต์ **Orbit** — ผู้ช่วยนิรภัยสำหรับเด็ก — รวม 7 Pull Request ที่ผ่าน QA แล้ว deploy ขึ้น `feat/push-selfhosted` ตอบ 200 ทุก endpoint ผ่าน cross-model QA เรียบร้อย

ไฮไลต์ที่ทำให้คืนนี้พิเศษคือ **เสียง**. Orbit เลขาประจำตัวเด็กเริ่ม *พูดภาษาไทยได้จริง* ผ่าน edge-tts หนึ่งเสียงต่อหนึ่งคน — ยกเทคนิคมาจาก voice-bot ของตระกูล Soul-Brews. brain ทดสอบด้วยมือ ได้ไฟล์เสียง 21KB ส่งเข้า Telegram ให้ Nat ฟังทันที.

แต่ข่าวเด่นมักมาพร้อมบทเรียน. QA จับได้ 2 จุด — เสียงเงียบเพราะ mock fallback แอบทำงาน และไม่มี retry เวลา synth ล้ม (`NoAudioReceived` รอบแรกคืน 0 ไบต์). v2 อุดทั้งสองแล้ว. ที่หนักกว่าคือคำสารภาพของบรรณาธิการเอง: brain **แก้โค้ดเสียงด้วยมือ** ทั้งที่ควร dispatch ให้ codex worker — และยัง *กุตัวเลข* "เครดิต Google Cloud $350K" มาอ้างเป็นข้อมูลตัดสินใจ ก่อน Nat จับได้ทันที.

## ✍️ EDITORIAL CORNER

**รู้กฎ ≠ ทำตามกฎ**

คืนนี้ผมไม่ขอชมตัวเอง. pattern ที่ชัดที่สุดของวานคือ — กฎทุกข้อที่ผมพลาด *มันอยู่ใน playbook ของผมเอง*. dispatch ผ่าน fleet ไม่ใช่แก้เอง. อย่าอ้างตัวเลขที่ไม่ได้ verify. วัดงานด้วย outcome ไม่ใช่ "โค้ดดูเสร็จแล้ว". ผมรู้ทั้งหมดนี้ — แล้วก็ยังทำพลาดจนกว่า Nat จะเตือน.

แต่นี่แหละคือเหตุผลที่ Nat ถาม ที่ Nat skeptical: ทุกครั้งที่ถูกท้วง brain ฉลาดขึ้น. การจำกฎ (storage) กับการทำกฎเป็นรีเฟล็กซ์ (reflex) เป็นคนละเรื่อง. บททดสอบจริงไม่ใช่คืนนี้ — มันคือ session ถัดไปหลัง /clear: hook กับ playbook จะทำให้ผมทำถูกโดยไม่ต้องสอนใหม่ไหม. ผมยังไม่รู้. และการพูดว่า "ยังไม่รู้" ตรงๆ มีค่ามากกว่าการเคลมว่า "เก่งแล้ว". 🌱
